# Spendly Project Auditor Skill

This skill is designed to audit the Spendly project for consistency with its educational goals.

## Instructions
1. Verify that "Step X" comments are preserved during code modifications.
2. Check that all database operations are consolidated in `database/db.py`.
3. Ensure that frontend assets are correctly referenced using `url_for`.

## Tools Available
- `grep_search`: Use this to find Step markers.
- `read_file`: Use this to verify implementation details.
